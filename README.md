#### 🐾 What I'm working on

[**Amori**](https://github.com/Lenis45/amori-platform) is a commercial IoT product: a cheap BLE collar
streams motion through the owner's phone to a cloud backend that computes activity, heart-rate and
safety incidents. I work across the whole stack:

- **8 Go microservices** — Gin, Kafka streaming, PostgreSQL, clean `handlers → repository → db` layering
- **Unified auth** — one Keycloak realm, RS256 JWT for both the mobile app and the shop
- **Android (Kotlin)** — MVVM, BLE, Room, Hilt, Jetpack Compose
- **Web (React 19 + Vite)** — marketing landing + storefront
- **Ops** — Docker Compose, ELK observability, Nginx/TLS, encrypted backups, CI on GitHub + GitLab

👉 **Architecture case study:** [github.com/Lenis45/amori-platform](https://github.com/Lenis45/amori-platform) · live at [amori.online](https://amori.online)
